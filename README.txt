=======================================================================
                              Blaise 0.9                                  
=======================================================================

1.  Introduction

    Blaise is an open-source object-oriented scripting language.


2.  Files included in this distribution

    Bin\                 - Blaise Binaries 
      BlaiseCLI.exe      - Blaise Command Line application
      BlaisePrompt.exe   - Blaise Prompt application
      Blaise.dll         - Blaise runtime library
      BlaiseIDE.exe      - Blaise IDE (experimental)

    Doc\                 - Blaise Documentation
      Blaise-UG.txt      - Users Guide

    Lib\                 - Standard Blaise libraries

    Examples\            - Blaise Examples

    Source\              - Blaise Source code
    Source\CLI           - Source for Blaise Command Line Inteface application
    Source\DLL           - Source for Blaise DLL
    Source\Prompt        - Source for Blaise Prompt application
    Source\Units         - Source for Blaise units.
    Source\Fundamentals3 - Source for Fundamentals3 (utility libraries).
    

3.  Version history

    * 31-10-2003 v0.90 Beta 1   
      First public release.


4.  Compiling the source code

    The source code requires Delphi 5 or later to compile.
